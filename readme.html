tsv2csvの概要（執筆中）

TSVをCSVに変換するツールです。 Excelでセル範囲をコピーするとクリップボードにTSVとしてコピーされます。 変換処理はすべてブラウザ内部で行われますのでサーバにデータが送信されることはありません。

Excelのセルは二重引用符、改行、タブを含む場合があります。 セルがタブも改行も含まなければ、二重引用符は特別な扱いを受けることなくTSVに現れます。 セルがタブか改行を含む場合、セルの内容はTSVにおいて二重引用符で囲まれ、 さらにそのようなセルに含まれていた二重引用符は連続する二重引用符としてTSVに現れます。 たとえば例４のExcelシートにおいて一行目と二行目は異なる内容であるにもかかわらず、 TSVにおいては同じ文字列になり、自動的に判別することができません。 そのためTSVにおいて二重引用符に遭遇した場合の取り扱いを指定する必要があります。 分析結果が期待した結果と異なる場合は、以下のオプションを変更して再度分析してください。
